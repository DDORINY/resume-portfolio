## Main Project
**LMS Management System**
Python + MySQL 기반 학습관리 시스템(LMS) 팀 프로젝트입니다.

**주요 기능**
- 회원 관리 (가입 / 수정 / 비활성화)
- 로그인 / 세션 인증
- 성적 관리
- 게시판
- 상품 관리
- 관리자 권한 분리

**Architecture**
- Layered Structure (Controller / Service / Repository)
- Service 클래스 단위 기능 분리
- DB 연결 모듈 분리
- 역할(Role) 기반 접근 제어
- 모듈화된 CRUD 설계

**Technical Points**
- Python OOP 기반 도메인 모델
- MySQL 스키마 설계 및 관계 모델링
- DictCursor 기반 데이터 처리
- Soft Delete 패턴 적용
- CLI → Web 확장 구조 고려
